Order to learn sorting algorithms:
                    Time Complexity     Space Complexity
1. Bubble Sort      O(n^2)              O(1)    
2. Selection Sort   O(n^2)              O(1)
3. Insertion Sort   O(n^2)              O(1)
4. Merge Sort       O(n log(n))         O(n)
5. Quick Sort       O(n log(n))         O(log(n))
